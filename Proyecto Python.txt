Proyecto de programador Python segundo semestre 2019

Tenemos como cliente una tienda de discos.

En dicha tienda estará registradas las disqueras, los
artistas y los discos. Un artista puede tener muchos discos, y un disco puede tener más de un
artista. Los discos sólo pueden tener una disquera.

Nuestro sitio debe de tener un modelo que represente esta realidad. El aspecto del sitio también es
importante.

Tendrá que tener una vista de búsqueda, otra de contacto, y la posibilidad de registrarse en el sitio.

La base de datos se puede desarrollar en sqlite3 nativo en Django o en MySQL, el alumno deberá
además hacer algunas vistas ya sea sobre la base de datos, o vistas para los contactos con el sitio. Se
está abierto a vistas que el alumno entienda convenientes y no sean ninguna de las anteriores.

El proyecto se desarrollará en Python utilizando Django, y se podrá utilizar todas tipos de
herramientas como CSS, javascript, contenidos estáticos como imágenes, etc. Estos estarán
definidos en settings.py.
